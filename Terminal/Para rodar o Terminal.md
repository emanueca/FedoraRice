Para rodar o Terminal, instale o Fastfestch ou neofestch (link que vai rederecionar para como instalar os pacotes do .config). Após isso, use um terminal de sua preferencia e instale os dois arquivos, após isso:

# 1. Criar o diretório
mkdir -p ~/.config/fastfetch

# 2. Mover os arquivos que abaixou
mv ~/Downloads/config.jsonc ~/.config/fastfetch/
mv ~/Downloads/thinkpad.txt ~/.config/fastfetch/

# 3. Testar
fastfetch
